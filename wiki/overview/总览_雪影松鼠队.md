## 角色分工

```
启动层：雪影娃娃 ──⭐场地战术起点──→ 赤子之心(-3能耗)
                │
                ├──首领化────→ 冻结×2 + 能耗+1 全员
                ├──赤子之心──→ 自身-3能耗buff（永久）
                └──击鼓传花──→ 传递给下一个入场精灵
                        │
                        ├──→ 蹦床松鼠（最高优先级：永久-3 → 大C）
                        └──→ 翠顶夫人/圣羽翼王（次要）

⚠️ 雪影娃娃威胁起点（敌方可能打断赤子之心的手段）：
| 威胁类型 | 来源 | 应对 |
|----------|------|------|
| **偷袭** | 敌方先手宠物/迅捷技能 | 雪影中低速(90)，首发冻结争取时间 |
| **迅捷** | 圣羽翼王水刃、翠顶夫人飞羽等 | 首领化后全员能耗+1压制 |
| **愿力** | 高爆发必杀技秒杀 | 翠顶夫人开力量增效诱骗消耗CD |
| **钢铁洪流** | 帕帕斯卡1号位160威力 | 非直接威胁（我方宠物） |
| **高伤技能** | 各类克制技能 | 风墙/肉盾联防 |

输出层：
  ├─ 主炮台：蹦床松鼠（赤子之心后，0费追打6连击）
  ├─ 副炮台：圣羽翼王（迅捷0费水刃轮切）
  └─ 捕获手：帕帕斯卡（迅捷偷袭3倍秒杀工兵）

联防层：
  ├─ 翠顶夫人：风墙挡技 + buff接力
  └─ 尖嘴狐仙：肉盾 + 印记处理

兜底层：
  └─ 蹦床松鼠（无buff时作为拦截位抗伤害）
```

## 联动链路

### 主链路：雪影 → 蹦床松鼠（赤子之心传递）

> 雪影娃娃 = **场地战术起点**（Originator），蹦床松鼠 = **终点**（Beneficiary）

```
雪影娃娃首发 → 首领化（冻结×2 + 能耗+1 全员）
  → 安全回合打出赤子之心（自身-3能耗）
  → 击鼓传花（传递buff给下一位）
  → 下一位选择：
      ├─ 最高优：直接传蹦床松鼠 → 永久-3能耗 → 大C启动
      └─ 次优：传翠顶夫人 → 力量增效 → 再传主力
```

### 辅助链路：捕获反工兵

```
敌方有工兵（飞羽/化蝶/晒太阳）？
  → 保帕帕斯卡状态（轴承支撑管理能耗）
  → 第一发钢铁洪流正常用掉（1号位+90威力=160，不浪费）
  → 技能轮转后偷袭击回到1号位
  → 上蹦床松鼠施压 → 敌方切工兵清buff
  → 先读切帕帕斯卡 → 迅捷（1号位加成）+ 偷袭 ×3倍 = 255威力 → 秒杀工兵
  → 蹦床松鼠安全享受场地效果
```

### 愿力诱饵链路

```
雪影需要安全开赤子之心 → 怕愿力秒杀
  → 翠顶夫人大胆开力量增效（愿力高发点）
  → 敌方愿力杀翠顶 → 消耗一次愿力CD
  → 雪影娃娃下一轮安全开赤子之心
  （冗余进攻：用翠顶换雪影吃愿力）
```
