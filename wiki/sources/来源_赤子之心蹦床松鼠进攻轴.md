# 来源：赤子之心蹦床松鼠进攻轴

## 摘要

**进攻轴组成（3精灵）：**

| 角色 | 宠物 | 技能组 |
|------|------|--------|
| **场地起点** | 雪影娃娃 | 暴风雪 / 冰墙 / **赤子之心(-3能耗)** / 击鼓传花 |
| **场地终点** | 蹦床松鼠 | 热身运动(连击) / **追打**(75威力) / 休息回复 / 吓退 |
| **捕获手** | 帕帕斯卡 | **钢铁洪流**(1号位+90=160) / 轴承支撑 / **偷袭击**(应对状态×3) / 倾泻 |
| 迅捷顺序：钢铁洪流 → 偷袭 → 偷袭 → 倾泻（4技能循环） |

**三角色关系：**
- 起点→终点：雪影打出赤子之心+击鼓传花 → 传递给蹦床松鼠 = 无解大C
- 工兵→终点：敌方飞羽/晒太阳清掉蹦床buff = 瞬间反转局势
- 捕获手→工兵：雪影/帕帕斯卡先读捕获敌方工兵 = 保护终点安全

**核心内容覆盖：**
1. **起点篇** — 纯小子流轮切大法 + 5类威胁点识别 + 博弈优劣方判断
2. **终点篇** — 工兵类型分析（飞羽高威胁 > 化蝶中威胁 > 其他晒太阳低威胁）
3. **捕获篇** — 利用松鼠压力制造先读机会，雪影低风险低回报 vs 帕帕斯卡高风险高回报
4. **反先读** — 高手可能反读松鼠入场，需将捕获前置
