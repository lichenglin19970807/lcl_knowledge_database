# 来源：PVP 非冗余战术

> 笔记原文为**联攻**系列第二篇，聚焦**非冗余战术**——当不存在1看2/2看3局面时的进攻方案。

## 内容概要

| 模块 | 核心观点 |
|------|----------|
| **非冗余 vs 冗余** | 冗余=重复对位关系突破 counter；非冗余=利用场上压力+先读创造局部优势，适用范围更广 |
| **炮台战术** | 识别高压力宠物→迫使对方切人→先读切换安全上场的炮台；多炮台变体：A压出C→切B吃C |
| **捕获战术** | A 压力逼出 B → 先读 B 上场 → 迅捷 C 直接重创 B；B 往往是 A 的拦截位，重创后 A 可推队 |
| **场地战术** | 印记/永久buff 改变 counter 关系；三角色模型：起点（发起者）、终点（受益者）、工兵（清除者）|

## 核心区别

非冗余战术的效果取决于**创造出的场景是先读还是猜拳**——与敌我水平有关。高分段先读退化为猜拳时，非冗余战术效果下降。

## 关键案例

- **多炮台**：白金独角兽压出雪影娃娃 → 先读切火神 vs 雪影 → 安全叠 buff
- **捕获**：白金压出雪影 → 先读帕帕斯卡迅捷秒杀雪影 → 白金再上场无人可挡
- **场地起点被工兵 counter**：圆号鱼（起点）vs 尖嘴狐仙（工兵）→ 终点选迅捷翼王或火系反制工兵
