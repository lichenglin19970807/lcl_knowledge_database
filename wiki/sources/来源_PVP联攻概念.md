# 来源：PVP 联攻概念

> 笔记原文归类为**战术**中的**联攻（进攻思路）**，分为冗余战术和非冗余战术。

## 内容概要

| 模块 | 核心观点 |
|------|----------|
| **进攻冗余前提** | 我方存在大C（能推队或1换2），但被敌方某只宠物 counter/check 无法启动 |
| **定向施压** | 多个大C共享同一 counter 位 → 1看2 / 2看3 → 硬换消耗 counter → 另一个大C 推队 |
| **控制战术** | 无复数大C时用控制手（如雪影娃娃冻结）逐步处理 counter 位 |
| **地雷战术** | 作者标记为待深入研究，暂不展开 |

## 关键案例

- **1看2**：白金独角兽 + 蹦床松鼠 vs 雪影娃娃（唯一 counter）→ 定向施压突破
- **2看3**：黑猫 + 白金 + 蹬床 vs 圣剑-X + 雪影娃娃（双拦截位）
- **控制场景**：蹦床松鼠（大C）+ 雪影娃娃（控制手）vs 燃薪虫
