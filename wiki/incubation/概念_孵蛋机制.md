# 概念：孵蛋机制

## 基本条件

| 条件 | 说明 | 必要性 |
|------|------|--------|
| **蛋组相同** | 公母必须属于同一蛋组才能生蛋 | ✅ 必要 |
| **目标精灵为母亲** | 孵出精灵随母系（如母雪影娃娃 → 蛋一定是雪影娃娃） | ✅ 必要 |

## 3v 遗传机制

**3v** = 性格 + 3 个努力值项

- 父亲 **30%** 概率遗传其 3v
- 母亲 **30%** 概率遗传其 3v
- 剩余 40% 随机生成

> [!tip] 提高遗传率的方法
> 让父母双方的 **3v 完全相同** → 无论遗传父还是母，结果都一样，实际有效遗传率大幅提升。

## 迭代代数系统

```
1代 (野生/初始) → 同3v配对 → 2代 → 同3v的2代配对 → 3代 → ...
```

**规律**：
- 用同 3v 的公母孵化出相同 3v 的精灵 → **2 代**
- 2 个同 3v 的 2 代个体配对 → **3 代**
- **代数越高 → 孵出的精灵 3v 与父母相同的概率越大**

## 异色遗传

- **父母中至少一方异色** → 孵出异色概率提升
- 双方异色 → 概率进一步提升（具体倍率待确认）

## 孵蛋优先级决策

按重要性排序：

| 优先级 | 条件 | 原因 |
|--------|------|------|
| P0 | 目标精灵 = 母亲 + 蛋组匹配 | 必要条件，无可妥协 |
| **P1** | **父母至少一方异色** | 异色权重最高，优先于3v匹配 |
| P2 | 父母 3v 相同 | 提高3v遗传效率 |
| P3 | 迭代代数越高越好 | 代数越高遗传越稳定 |

> [!warning] P1 vs P2 权衡
> 如果异色和3v匹配不能同时满足，**优先保证异色（P1）**。
> 对当前账号来说，异色的价值高于完美3v。
