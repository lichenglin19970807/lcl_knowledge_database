# 概念：对阵策略_雪影vs音速犬

## 对局背景

| 项目 | 内容 |
|------|------|
| 我方 | [[overview/总览_雪影传说队]] — 固定雪影娃娃首发 |
| 敌方 | [[entities/对手_音速犬]] — 火系高速物攻，可选首发 |
| 核心矛盾 | 音速犬入场即+100%物攻 + 热身4倍机制克制我方风墙战术 |

## 前提条件

1. **雪影娃娃固定首发** → 第一回合首领化（给敌方上2层冻结+能耗+1）
2. **敌方选择音速犬首发** → 入场获得物攻+100%
3. 首领化结束后，雪影娃娃需要**切人**（无法扛住音速犬输出）

## Case 分析

### Case 1：常规思路（已验证为陷阱）

```
T1: 雪影娃娃首领化 → 敌方音速犬入场(+100%物攻)
T2: 我方切 翠顶夫人 → 放 迅捷风墙（防御状态技能）
     ↓
     敌方音速犬使用 热身
     ↓
     ⚠️ 风墙被判定为"防御类状态技能"
     → 热身的下一次攻击威力变为 4倍！
     ↓
T3: 音速犬攻击（热身后 4倍威力）
     → 无人能扛 → 必送一只宠物
     → 基本判负
```

> [!danger] 为什么这是陷阱？
> 翠顶夫人的风墙原本是用来挡高威胁技的。
> 但风墙是"防御类状态技能"，恰好触发热身的4倍条件。
> **用风墙去挡热身 = 给音速犬递刀子。**

### Case 2：当前最优解（圣羽翼王换入）

```
T1: 雪影娃娃首领化 → 敌方音速犬入场(+100%物攻)
T2: 我方切 圣羽翼王 → 触发迅捷
     ↓
     ┌─ 分支 A：音速犬出 热身 ─┐
     │  水刃"应对"到状态技能      │
     │  → 水刃费用变为 0          │
     │  → 迅捷 + 0费水刃          │
     │  → 打掉音速犬 50%+ 血量    │
     │  → ✅ 大赚                │
     └───────────────────────────┘
     ↓
     └─ 分支 B：音速犬出 攻击技能 ─┐
        （跌落/当头棒喝/灼烧/火云车）
        → 圣羽翼王(132血/128物防) 承受一次攻击
        → 不会秒杀（音速犬虽高攻但非克制翼系）
        → 迅捷水刃反打一换一或换二
        → 🟡 可接受              │
        └───────────────────────────┘
```

## 决策树

```
雪影娃娃首领化完成，需要切人
│
├── 切 翠顶夫人 放风墙？
│   └── ❌ 陷阱！若音速犬热身 → 4倍后续攻击 → 基本输
│
├── 切 圣羽翼王 打水刃？  ← 推荐 ✅
│   ├── 敌方出热身 → 水刃0费 → 打掉50%+ → 大赚
│   ├── 敌方出攻击 → 肠一发但不死 → 可接受
│   └── 敌方出当头棒喝（赌我换人）→ 160威力打过来 → 有风险
│
├── 切 圣剑-X 吃伤害？
│   └── 三围高(174/173/173)大概率扛得住 → 但速度65后手
│       → 回击一轮但循环可能不够
│
└── 不切人，雪影娃娃硬抗？
    └── ❌ 物防66 + 冰萌被火克制 = 秒杀
```

## 各方案评估

| 方案 | 最佳情况 | 最差情况 | 推荐度 |
|------|----------|----------|--------|
| 切翠顶风墙 | 挡住一次攻击 | **触发4倍热身→崩盘** | ❌ 禁止 |
| **切圣羽水刃** | **0费打半血以上** | **吃一发攻击不死** | **⭐⭐⭐⭐⭐** |
| 切圣剑-X | 扛住+反打一轮 | 连续被压制循环断档 | ⭐⭐⭐ |
| 送掉雪影娃娃 | 无价值牺牲 | 无价值牺牲 | ❌ |
