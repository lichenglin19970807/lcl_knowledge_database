# 概念：猜拳与先读

## 猜拳（Rock-Paper-Scissors）

### 基础猜拳三角

洛克王国最基础的技能类型克制链：

```
攻击 ──克制──→ 状态
状态 ──克制──→ 防御
防御 ──克制──→ 攻击
```

在不考虑双方习惯的情况下，纯猜拳的**输赢概率为 50:50**。

### 广义猜拳

猜拳不仅限于技能类型选择。当双方**信息识别能力处在同一水平**时，以下决策都属于猜拳：

- 切人时机
- 技能选择（多个可用技能时）
- 强化/输出节奏判断

### 实战案例：音速犬 vs 机械系

| 音速犬选项 | 机械系应对 | 结果 |
|-----------|-----------|------|
| 打火系技能（克制机械） | 留场硬扛 | 音速犬占优 |
| 打火系技能 | 切水系抵抗火系 | 损失一回合输出 |
| **用当头棒喝** | 切水系宠物 | **当头棒喝触发切人特效 +100威力 → 秒杀水系** |
| 用当头棒喝 | **留场抵抗当头棒喝** | **音速犬浪费高伤机会** |

> 当头棒喝：耗能3，物攻，普通，80威力；若敌方本回合更换精灵，威力 +100。

当双方对上述所有可能性都思考到位时，这就是**猜拳**——谁也压不了谁。

### 猜拳的本质效应

> **猜拳会拉近高手和低手之间的胜率。**
>
> 无论你属于什么水平，当你大部分时间都在猜拳时，理论上胜率就接近 **50%**。

## 先读（Read）

### 定义

> **先读 = 当你的信息、场景识别能力高于对手时，你能够预判对手的操作，从而大幅提升猜赢概率。**

### 核心要素：信息差

先读不是玄学——它建立在**可利用的信息不对称**之上：

1. **技能携带信息差** — 对方不知道你带了什么技能
2. **套路熟悉度信息差** — 你知道对方的常用打法，对方不了解你的
3. **场景识别能力差距** — 同样的局面你能看到更多可能的选择分支

### 续上例：音速犬当头棒喝的先读场景

前提条件：**音速犬带当头棒喝是小众选择（约1%）**

```
机械系的视角：
  "音速犬有火系技能，check 我" → 场上压力大 → 大概率切水系

音速犬（你）的视角：
  "他不知道我有当头棒喝"
  "他大概率会切水系抵抗我的火系技能"
  → 先读：直接用当头棒喝 → 秒杀切入的水系宠物
```

这就是**先读**——利用信息差将猜拳变成确定性操作。

## 先读 vs 猜拳：分水岭

| 维度 | 猜拳 | 先读 |
|------|------|------|
| **信息水平** | 双方相当 | 己方 > 对方 |
| **胜率** | ≈ 50% | >> 50% |
| **本质** | 概率博弈 | 信息优势变现 |
| **稳定性** | 不稳定 | 相对稳定 |

## 上分路径理论

PVP 排位上分的底层逻辑：

```
┌─────────────────────────────────────────────┐
│  低分段                                      │
│  你的信息/场景识别 > 同分段选手               │
│  → 更多先读场景 → 胜率 > 50% → 上分          │
├─────────────────────────────────────────────┤
│  所属水平分段                                │
│  信息/场景识别 ≈ 对手                         │
│  → 先读退化为猜拳 → 胜率 → 50%（瓶颈）        │
├─────────────────────────────────────────────┤
│  更高分段                                    │
│  需继续提升信息/场景识别能力                   │
│  → 重新建立信息优势 → 新的先读场景 → 继续上分  │
└─────────────────────────────────────────────┘
```

**核心结论**：上分的根本途径不是练操作，而是**提升信息处理和场景识别能力**。
