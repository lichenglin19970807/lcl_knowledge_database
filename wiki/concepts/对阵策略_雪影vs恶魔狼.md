# 对阵策略_雪影vs恶魔狼

> 恶系首领化收尾宠 · 力竭叠攻的应对思路

[[entities/宠物_恶魔狼|📖 实体档案]] | [[sources/来源_雪影vs恶魔狼|📋 来源摘要]]

## 核心矛盾

| 维度 | 数据 |
|------|------|
| [[entities/宠物_恶魔狼\|恶魔狼]] 出场时机 | **最后一个**（首领化收尾位） |
| 此时双方力竭数 | 通常已积累多个 → 双攻已大幅叠加 |
| 我方速度优势 | **仅[[entities/宠物_圣羽翼王\|圣羽翼王]](130) > 120** |
| 恶魔狼属性克制 | 被光/武/虫/萌克制，但我方无纯系对应输出核心 |

## 应对策略

### 策略一：保 [[entities/宠物_雪影娃娃\|雪影娃娃]] 满血 ✅ 最高优先级

> 雪影娃娃最好保持**满血**状态，不会被恶魔狼秒杀。

| 要点 | 说明 |
|------|------|
| 为什么保满血 | 恶魔狼双攻叠加后伤害极高，满血雪影娃娃是唯一能硬抗一轮的单位 |
| 如何做到 | 前中期避免雪影娃娃被换出消耗，尽量用其他宠物轮换承伤 |
| 价值 | 雪影娃娃存活 = 冻结控制 + 能耗限制，为反杀创造条件 |

### 策略二：保 [[entities/宠物_圣羽翼王\|圣羽翼王]] 费用 ⚠️ 关键资源管理

> 最好能留到 **0费水刃**，否则需预留 **4费** 打迅捷水刃。

| 场景 | 费用需求 | 说明 |
|------|---------|------|
| 理想情况 | **0费**（应对到状态技能） | 迅捷 + 0费水刃 → 连续输出压制 |
| 一般情况 | **4费** | 正常水刃费用 + 迅捷触发 |
| 最差情况 | 费用不足 | 圣羽翼王无法有效输出 → 失去唯一速度优势单位 |

### 为什么只有圣羽翼王能处理？

```
我方速度线 vs 恶魔狼(120)
│
├── 圣羽翼王(130) ──→ ✅ 先手权在手
│   ├── 有费用 → 水刃压制 / 击杀
│   └── 无费用 → 废掉
│
├── 音速犬(120) ──→ = 同速，随机先后（不可靠）
│
├── 其他全部 <120 ──→ ❌ 后手
│   └── 后手吃恶魔狼高额双攻一击 → 大概率被秒
│
└── 结论：圣羽翼王是唯一的可靠counter
```

## 决策树

```
对局进入中后期，敌方可能派恶魔狼收尾
│
├── 雪影娃娃是否满血？
│   ├── ✅ 是 → 继续保护，作为最后的防线
│   └── ❌ 已受伤 → 尽量避免再暴露，准备送其他宠
│
├── 圣羽翼王是否有足够费用？
│   ├── ✅ 有（理想0费 / 至少4费）→ 留待恶魔狼出场时切出
│   └── ❌ 不足 → 优先在前期找机会回费或触发0费
│
└── 双方损宠情况？
    ├── 己方损宠少 → 控制交换节奏，减少对方力竭叠加
    └── 己方损宠多 → 恶魔狼即将超神 → 必须速战速决
```

## 总结

| 优先级 | 目标 | 操作 |
|--------|------|------|
| **P0** | 保雪影娃娃满血 | 避免前中期消耗 |
| **P1** | 保圣羽翼王费用 | 瞄准0费水刃，底线4费 |
| P2 | 控制己方损宠数量 | 减少恶魔狼特性叠加层数 |
