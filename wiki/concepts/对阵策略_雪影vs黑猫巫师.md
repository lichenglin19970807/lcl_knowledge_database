# 概念：对阵策略_雪影vs黑猫巫师

## 对局背景

| 项目 | 内容 |
|------|------|
| 我方 | [[overview/总览_雪影传说队]] |
| 敌方 | [[entities/宠物_黑猫巫师]] — 普通系核爆，0费技能秒杀级输出 |
| 核心矛盾 | 黑猫的两套套路（魔能爆彗星/羽化连击）都能稳定换掉1-2宠，我方仅圣剑-X能扛 |

## 前提认知

### 黑猫巫师的两种形态

| 流派 | 核心特点 | 主要威胁方式 | 铺垫需求 |
|------|----------|--------------|----------|
| **魔能爆彗星流** | 阵地一波带走 | 0费魔能爆 + 0费彗星连续核弹 | T1 博弈叠buff（魔法增效/嗜痛） |
| **羽化连击流** | 游游击轮切收割 | 迅捷连击（5段×20/25）+ 威力buff | 切人触发迅捷 → 羽化加速启动 |

> [!danger] 两套套路的共同点
> - **都能稳定换掉我方 1~2 只精灵**
> - **都需要铺垫回合**（不是无脑扔核弹）
> - **除圣剑-X外全员一击即死**

## 应对策略 — 分层防御

### 第一层：核心 counter — 圣剑-X

**圣剑-X 是对阵黑猫巫师的唯一真解。**

```
原因：
├── 生命174 / 物防173 / 魔防173 → 三围顶格
├── 魔能爆打不穿 → 即使满 buff 的核弹也未必秒杀
├── 彗星240威力也未必秒杀（衰减后）
└── 反过来：械斗 + 鸣沙陷阱 可以杀掉黑猫

操作：
T1: 看到/预判黑猫上场 → 直接切 圣剑-X
T2-T4: 按循环输出 → 械斗/鸣沙陷阱/啮合传递/齿轮扭矩
    └── 不需要特殊操作，硬抗+循环反杀即可
```

### 第二层：如果圣剑-X 无法保住

按优先级排序：

| 替代方案 | 可行性 | 分析 |
|----------|--------|------|
| **切雪影娃娃** | ⭐⭐⭐ | 魔防130尚可 + 冰萌被普通抵抗(0.5x) = 有一定生存率。但物防66怕物理连击 |
| **切尖嘴狐仙** | ⭐⭐⭐ | 肉盾三耐配置 + 火/冰双属性有抵抗面。魔防107偏低但有血量撑 |
| **切翠顶夫人放风墙** | ⭐⭐ | 迅捷风墙可以挡一次核弹。但风墙挡不住连击（多段判定）。且只有一次机会 |
| 送掉非核心宠物 | ⭐ | 如果上述都不行，宁可送翠顶/尖嘴也不要送龙息/圣羽 |

### 第三层：预防性策略（看到对方阵容有黑猫时）

```
1. 保圣剑-X 健康
   ├── 不要让圣剑-X 过早消耗血量
   ├── 不要让圣剑-X 对位不需要对位的宠物
   └── 它就是留给黑猫的

2. 控制对方资源
   ├── 通过雪影娃娃的冻结debuff压制全队
   └── 让黑猫没有舒服的铺垫回合

3. 信息博弈
   ├── 观察对方带的哪套技能组
   │   ├── 有彗星 = 魔能爆彗星流
   │   └── 有羽化加速/午夜噪音 = 羽化连击流
   └── 根据流派调整应对节奏
```

## 决策树

```
敌方场上出现黑猫巫师？
│
├── 圣剑-X 存活且健康？
│   └── ✅ **直接切圣剑-X** → 硬抗+循环反杀（最优解）
│
├── 圣剑-X 已阵亡/残血？
│   ├── 雪影娃娃可用？
│   │   └── 切雪影（魔防130 + 抵抗普通0.5x）
│   ├── 尖嘴狐仙可用？
│   │   └── 切尖嘴狐仙（肉盾面板兜底）
│   ├── 翠顶夫人可用？
│   │   └── 切翠顶放风墙（挡一次核弹，赌连击流）
│   └── 都不可用？
│       └── 送掉当前场上最不值的宠物，保住核心
```

## 对比：黑猫巫师 vs 音速犬

| 维度 | 音速犬 | 黑猫巫师 |
|------|--------|----------|
| 威胁类型 | 高速物攻爆发（热身4倍） | 核爆法术输出（0费魔能爆/彗星） |
| 速度 | **120**（快） | **70**（慢） |
| 我方 counter | 圣羽翼王（水刃0费） | **圣剑-X（唯一能扛的）** |
| 反制难度 | 中等（博弈热身） | **极高（基本只能硬抗）** |
| 怕什么 | 水系克制 / 血薄 | 武系克制 / 速度慢 |
| 对我方杀伤范围 | 大部分（除圣羽外危险） | **极大部分（除圣剑外必死）** |
| 给我方的反应窗口 | T1 就要决策 | **有铺垫回合但核弹无法规避** |

> [!conclusion] 结论
> 音速犬是"高速刺客"，可以通过正确的首轮决策 counter。
> 黑猫巫师是"低速核弹"，几乎没有操作空间——**要么圣剑-X在，要么崩盘**。
>
> **黑猫巫师的威胁等级 > 音速犬。**
