# 概念：状态_冻结

## 定义

**冻结**是洛克王国世界中的持续性伤害+死亡判定型控制效果，是雪影传说队的核心 debuff 之一。

## 效果详解

| 项目 | 规则 |
|------|------|
| **伤害方式** | 每回合扣除 **5% 当前生命**（不是最大生命） |
| **死亡条件** | 若当前生命值 ≤ 冻结累计伤害比例 → **直接死亡** |
| **免疫属性** | **冰系宠物完全免疫冻结** |
| **持续时间** | 下场后**不消除**，持续挂在原精灵身上 |
| **解除方式** | 一般只有水系技能"**洗礼**"可以清除 |

## 在阵容中的应用

### 雪影娃娃 — 冻结核心施加者

详见 [[entities/宠物_雪影娃娃]]

- **普通形态**：攻击使敌方获得冻结 → 附带全技能能耗+1
- **首领化形态**：入场即给敌方上 **2 层冻结** + 能耗+1

### 与其他效果的联动

- 冻结 + 能耗+1（雪影娃娃特性）→ 对方既掉血又缺蓝，双重压制
- 冻结持续不掉出 → 对方无法通过换宠规避
- 只有水系洗礼能解 → 限制对方应对手段
