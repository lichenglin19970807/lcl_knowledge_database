# 概念：属性克制机制

## 定义

属性克制系统是《洛克王国世界》PvP 对战的核心战斗机制之一。攻击招式的**攻击属性**与被攻击方的**自身属性（单或双）**之间存在克制或抵抗关系，直接影响最终伤害输出。

## 计算规则

### 单属性 vs 单属性

| 关系 | 伤害倍率 | 说明 |
|------|----------|------|
| **克制** | ×2.0 | 攻击属性克制防御属性 |
| **抵抗** | ×0.5 | 攻击属性被防御属性抵抗 |
| **无关系** | ×1.0 | 无特殊修正 |

### 双属性目标（重点）

当被攻击方为**双属性宠物**时：

| 场景 | 伤害倍率 | 示例 |
|------|----------|------|
| 攻击属性克制**两个**属性 | **×3.0** | 火系技能打 草+冰 宠物 |
| 攻击属性抵抗**两个**属性 | **×0.5** | 火系技能打 水+岩 宠物 |
| 克制一个 / 抵抗一个 | ×1.0 | 相互抵消 |
| 克制一个 / 无关系 | ×2.0 | 正常克制 |
| 抵抗一个 / 无关系 | ×0.5 | 正常抵抗 |

> [!warning] 关键记忆点
> - **双倍克制 ≠ 4 倍，而是 3 倍**
> - **双倍抵抗 ≠ 0.25 倍，而是 0.5 倍**
> - 这意味着双属性宠物的极端情况被"压平"了

### 计算流程

```
1. 确定攻击招式属性 A
2. 确定被攻击方属性 B 和 C（双属性时有两个）
3. 分别查表：A→B 的关系，A→C 的关系
4. 综合倍率：
   - 都克制 → 3x
   - 都抵抗 → 0.5x
   - 一克一抵 → 1x
   - 其他 → 正常对应倍率
```

## 在 PvP 中的应用

- **选宠阶段**：根据对方阵容预判，选择克制覆盖面广的宠物
- **换宠决策**：利用属性切换获得抵抗优势（0.5x 减伤显著）
- **配招策略**：本属性技能不一定最优，有时需覆盖盲区
- **双属性宠物价值评估**：双属性可制造"免疫环"，但也存在双重弱点风险
