# 对阵策略_雪影vs蹦床松鼠

> 蹦床松鼠的减费 buff 传递与追打连击应对思路

[[entities/宠物_蹦床松鼠|📖 实体档案]] | [[sources/来源_雪影vs蹦床松鼠|📋 来源摘要]]

## 核心威胁分析

| 威胁层级 | 内容 |
|----------|------|
| **直接威胁** | 热身运动(连击+3) + 追打(3连击) = **6 连击**普通系魔法伤害 |
| **间接威胁（更危险）** | 通过赤子之心+击鼓传花传递**永久 -3 费 buff**给队友 |

## 应对策略

### 方案 A：[[entities/宠物_圣羽翼王\|圣羽翼王白嫖]] ✅ 最优

> 用圣羽翼王**白嫖 0 费水刃**

- 圣羽翼王触发迅捷 → 水刃应对到状态技能时费用变为 0
- 即使蹦床松鼠有永久减费 buff，0 费水刃依然高效
- 速度 130 > 蹦床松鼠 80，稳定先手压制

### 方案 B：[[entities/宠物_翠顶夫人\|翠顶夫人清 buff]] ⚠️ 有条件

用翠顶夫人的**迅捷飞羽**清除蹦床松鼠的 buff 时需注意：

| 条件 | 结果 |
|------|------|
| 蹦床松鼠**无热身运动 buff** | ✅ 可以扛住追打（1-3 连击伤害可控） |
| 蹦床松鼠**有热身运动 buff** | ❌ 追打可达 **6 连击** → 翠顶夫人大概率被秒杀 |

### 方案 C：魔抗手消耗 ⭐ 辅助手段

当蹦床松鼠有热身运动 buff 时（翠顶不能硬上），先用**高魔防单位消耗其费用**：

| 可用单位 | 说明 |
|----------|------|
| [[entities/宠物_雪影娃娃]] | 魔防较高，可扛几轮 |
| [[entities/宠物_尖嘴狐仙]] | 火+冰肉盾面板 |
| [[entities/宠物_圣剑-X]] | 三围顶格(174/173/173)，最稳选择 |

> 目标：把蹦床松鼠的技能费用耗空 → 无法释放热身运动/追打 → 安全处理

## 决策树

```
遭遇蹦床松鼠（可能有永久减费buff）
│
├── 圣羽翼王可用？ ──→ ✅ 切圣羽白嫖0费水刃（最优解）
│
├── 需要用翠顶清buff？
│   ├── 蹦床松鼠有热身运动buff？ ──→ ❌ 不能切翠顶（会被6连击秒杀）
│   │   └── 改用魔抗手消耗：圣剑-X / 雪影娃娃 / 尖嘴狐仙
│   │
│   └── 无热身运动buff？ ──→ ✅ 可切翠顶清buff（能扛住追打）
│
└── 关键判断：是否已确认蹦床松鼠有热身buff？
    ├── 未确认 → 先观察或用低代价单位试探
    └── 已确认有 → 直接走消耗路线
```

## 各方案评估

| 方案 | 最佳情况 | 最差情况 | 推荐度 |
|------|----------|----------|--------|
| **圣羽翼王白嫖** | 0费水刃持续压制 | 被预判换人 | **⭐⭐⭐⭐⭐** |
| 翠顶清 buff（无热身） | 成功清除关键 buff | 敌方不依赖 buff | ⭐⭐⭐ |
| 魔抗手消耗 | 安全耗空对方费用 | 消耗过程中送掉一只宠 | ⭐⭐⭐（有热身时的唯一选择）|
