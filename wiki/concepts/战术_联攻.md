# 战术：联攻（进攻总论）

> **联攻 = 系统化的进攻思路框架。**
>
> 当我方大C被敌方 counter/check 无法直接启动时，通过不同策略突破拦截位，让大C获得推队机会。

## 分类总览

联攻分为两大类，根据**对位关系的重复性**来选择：

| 维度 | **冗余战术** | **非冗余战术** |
|------|-------------|---------------|
| **核心机制** | 利用**重复的对位关系**创造进攻冗余 | 利用**场上压力 + 先读**创造局部优势 |
| **前提条件** | 存在 **1看2** 或 **2看3** 局面 | 无特殊前提（1看1、2看2 均可） |
| **效果确定性** | 高 | 中（依赖先读 vs 猜拳） |
| **水平敏感度** | 较低 | **高**——高分段先读退化为猜拳 |
| **子分类** | 定向施压 / 控制 / [[concepts/战术_地雷|地雷]] | 炮台 / 捕获 / 场地 |
| **详见** | → [[concepts/战术_冗余进攻]] | → [[concepts/战术_非冗余]] |

## 选择逻辑

```
我方大C被敌方 counter/check 无法启动
│
├─ 对位关系中存在"重复"吗？
│   ├─ 1看2 或 2看3？ ──→ 冗余战术 ──→ [[concepts/战术_冗余进攻]]
│   │   ├─ 有复数大C？ → 定向施压（起点终点可互换）
│   │   ├─ 有控制手？   → 控制（固定起止点）
│   │   └─ 其他？       → 地雷（待研究）
│   │
│   └─ 1看1、2看2？    ──→ 非冗余战术 ──→ [[concepts/战术_非冗余]]
│       ├─ 有炮台？     → 炮台（压力换安全上场）
│       ├─ 有迅捷+明确拦截位？ → 捕获（迅捷重创切入位）
│       ├─ 有场地能力？ → 场地（印记改变counter关系）
│       └─ 都没有？     → 纯猜拳（胜率≈50%）
```
