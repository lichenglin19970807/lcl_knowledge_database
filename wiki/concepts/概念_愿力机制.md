# 概念：愿力机制

## 基础信息

| 项目 | 说明 |
|------|------|
| **威力** | 80（基础） |
| **攻击类型** | 取精灵**物攻/魔攻中较高者** |
| **属性** | = 精灵**血脉属性**（可自由修改 → **不可预测**） |
| **使用次数** | 敌方无首领化时 **2 次** |
| **冷却** | **3 回合** |

## 伤害计算

> [!danger] 应对状态加伤
>
> 愿力如果**应对到状态技能**，额外增加 **1.5×** 伤害倍率：
>
> | 场景 | 总伤害倍率 |
> |------|-----------|
> | 不克制 + 不应对状态 | **1.0×** |
> | 克制 + 不应对状态 | ~1.5~2.0×（视克制倍率） |
> | **不克制 + 应对状态** | **2.5×** |
> | **克制 + 应对状态** | **3.5×** ⚠️ |

这意味着：**只要敌方愿力应对到状态技能，即使不克制也能造成巨额伤害。**

## 核心博弈点

### 属性不可预测

愿力的属性 = 精灵血脉属性，而血脉可以随便改。
→ **你无法从精灵本身判断愿力的属性**
→ 敌方可能故意设置一个克制你当前上场精灵的血脉

### 逆属性战场信号

当敌方用**逆属性**精灵上场时，很可能带的是**克制我方精灵的愿力**。

> [!example] 实战案例
> [[entities/宠物_冰钻布鲁斯]]（冰系）对战机械系的[[entities/宠物_圣剑-X]]
> → 冰钻布鲁斯逆属性上场 → 很可能带**水愿力** → 水克机械 → 圣剑-X被秒

## 雪影传说队应对策略

### 1. 尖嘴狐仙 — 焚烧烙印风险

| 风险 | 说明 |
|------|------|
| 焚烧烙印是**状态技能** | 极易被敌方愿力"应对"到 |
| 应对后 2.5× 伤害 | 尖嘴狐仙被直接秒杀 |
| **对策** | 使用焚烧烙印前必须博弈预判敌方是否留有愿力次数 |

### 2. 圣剑-X — 轴承支撑回合风险

| 风险 | 说明 |
|------|------|
| 轴承支撑是**状态技能** | 转到该回合时只能聚能或打轴承支撑 |
| 行为可预测 | 敌方知道这回合你会出状态技 → 愿力针对 |
| **对策** | 圣剑转到轴承支撑回合时，**切换其他精灵上场**规避 |

### 3. 通用原则

| 原则 | 说明 |
|------|------|
| 记录愿力使用次数 | 2次用完 → 冷却3回合 → 安全窗口 |
| 无首领化才需要担心 | 敌方有首领化宠物时愿力机制不同 |
| 逆属性 = 警报 | 敌方逆属性上场优先考虑愿力风险 |
