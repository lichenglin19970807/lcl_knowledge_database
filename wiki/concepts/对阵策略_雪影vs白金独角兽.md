# 对阵策略_雪影vs白金独角兽

> 光系首领化折射推队的应对思路

[[entities/宠物_白金独角兽|📖 实体档案]] | [[sources/来源_雪影vs白金独角兽|📋 来源摘要]]

## 标准对策：[[entities/宠物_雪影娃娃\|雪影娃娃单防]] ✅

### 应对流程

```
遭遇白金独角兽
├── 切入雪影娃娃
│   ├── 首领化效果 → 独角兽能耗 +1（限制其低费技能）
│   └── 使用暴风雪 → 继续增加独角兽能耗
└── 结果：折射 + 追打秒不掉雪影娃娃 → 拖住 / 耗死
```

### 原理分析

| 机制 | 效果 |
|------|------|
| [[concepts/概念_机制_首领化\|雪影娃娃首领化]] | 入场即给敌方上能耗 +1 |
| 暴风雪 | 持续增加敌方能耗，抬高技能门槛 |
| 折射限制 | 能耗被抬高后，即使折射减耗也难以维持高频释放 |

**结论**：一般情况下，[[entities/宠物_雪影娃娃\|雪影娃娃]] 可以**单独防守**白金独角兽。独角兽的一次折射加追打无法击杀雪影娃娃。

## 特殊变种：椰浆布丁辅助 ❌ 无解

### 配置说明

部分对手会使用 **椰浆布丁** 作为辅助宠：

> **椰浆布丁特性**：离场后，更换入场的精灵获得 **双防 +20% 且免疫冻结**

### 为什么无解

| 维度 | 分析 |
|------|------|
| 冻结免疫 | 雪影娃娃的核心控制手段（冻结）完全失效 |
| 双防 +20% | 独角兽生存能力提升，更难被击杀 |
| 雪影针对失效 | 能耗增加手段仍在，但缺少冻结控制的配合，压制力大减 |

**当前状态**：面对带椰浆布丁的白金独角兽，**目前没有想到好的解决办法**。

## 决策树

```
遭遇白金独角兽
├── 标准配置？ ──→ ✅ 切雪影娃娃（首领化+暴风雪耗能拖住）
│   └── 确认：一次折射+追打杀不死雪影娃娃
└── 带椰浆布丁？ ──→ ❌ 目前无明确对策
    ├── 尝试雪影娃娃纯耗能？（冻结无效，效果存疑）
    └── 待研究：是否有其他counter路径
```
