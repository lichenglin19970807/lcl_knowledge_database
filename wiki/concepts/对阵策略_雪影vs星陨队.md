# 对阵策略_雪影vs星陨队

> 星陨队的星陨印记叠放与引爆应对思路

[[entities/宠物_怖哭菇|📖 核心实体：怖哭菇]] | [[sources/来源_雪影vs星陨队|📋 来源摘要]]

## 阵容背景

| 项目 | 说明 |
|------|------|
| 敌方 | **星陨队** — 以 [[entities/宠物_怖哭菇\|怖哭菇]] 为核心的阵容 |
| 杀伤链路 | 怖哭菇**叠星陨印记** → 圣羽翼王/龙息帕尔等**非幻系攻击引爆** |
| 我方挑战 | 如何处理已叠上的星陨印记？ |

## 应对流程

### Phase 1：[[entities/宠物_雪影娃娃\|雪影娃娃首发]] vs 怖哭菇

```
T1: 雪影娃娃首领化 → 敌方怖哭菇入场（开始被动叠印）
│
├── 攻击路线：暴风雪 ──→ ✅ 最稳
│   └── 直接输出伤害，不触发星陨（暴风雪为冰系但可能不触发？需验证）
│
└── 防御路线：冰墙 ──→ ⚠️ 有风险
    └── 若怖哭菇出错乱（应对状态）→ 叠3层星陨 → 后续更难处理
```

> **建议**：若敌方没有需要雪影娃娃专门处理的精灵，**暴风雪攻击最稳**。

### Phase 2：切 [[entities/宠物_尖嘴狐仙\|尖嘴狐仙]] — 印记处理核心

此时我方身上已积累若干层**星陨印记**，尖嘴狐仙是关键 counter。

#### 方案 A：焚烧烙印 — 直接清除

| 项目 | 数据 |
|------|------|
| 耗能 | 3 |
| 类型 | 状态 |
| 效果 | **驱散双方所有印记**，每驱散 1 层 → 敌方获得 **5 层灼烧** |

> 尖嘴狐仙将我方身上的星陨印记转化为对敌方的灼烧伤害，一举两得。

#### 方案 B：火焰护盾 — 博弈反制

| 项目 | 数据 |
|------|------|
| 耗能 | 2 |
| 类型 | 防御 |
| 效果 | 减伤 **70%**；若**应对攻击** → 敌方获得 **6 层灼烧** |

### Phase 2 博弈分析

```
尖嘴狐仙入场，身上有星陨印记
│
├── 敌方预判我打焚烧烙印 → 切圣羽翼王打迅捷水刃
│   ├── 水刃触发星陨引爆 → 尖嘴狐仙吃大量幻系伤害 💥
│   └── 且水刃费用未减少（没应对到状态）
│
└── 我方反预判：不打焚烧烙印，改打火焰护盾 🛡️
    ├── 水刃无法减费（火焰护盾是防御，不是状态）
    ├── 星陨印记被水刃引爆 → 但火焰护盾减伤70%吸收 🔰
    └── 圣羽翼王还吃到6层灼烧 🔥
```

## 决策树

```
遭遇星陨队
│
├── Phase 1：雪影娃娃首发 vs 怖哭菇
│   ├── 敌方无特别针对目标 → 暴风雪攻击 ✅（最稳）
│   └── 敌方有需针对目标 → 冰墙防御（风险：错乱叠3层印）
│
├── Phase 2：切尖嘴狐仙（处理星陨印记）
│   │
│   ├── 判断：敌方是否会预判焚烧烙印？
│   │   ├── 不太会 → 焚烧烙印直接清印 ✅
│   │   └── 可能会（切圣羽翼王）→ 火焰护盾反制 🧠
│   │
│   └── 方案 B 火焰护盾效果：
│       ├── 减伤70%吸收引爆伤害
│       ├── 给圣羽6层灼烧
│       └── 星陨印记被消耗掉（引爆后层数归零）
│
└── 关键资源：尖嘴狐仙必须存活到Phase 2
```

## 各方案评估

| 方案 | 最佳情况 | 最差情况 | 推荐度 |
|------|----------|----------|--------|
| 雪影暴风雪 + 尖嘴焚烧烙印 | 清印+灼烧敌方 | 被预判切圣羽引爆 | ⭐⭐⭐⭐ |
| 雪影暴风雪 + 尖嘴火焰护盾 | 反制成功，减伤+灼烧 | 敌方不切人，护盾浪费 | ⭐⭐⭐⭐（高阶博弈） |
| 雪影冰墙防御 | 可能挡住伤害 | 错乱叠3层星陨 | ⭐⭐ |
