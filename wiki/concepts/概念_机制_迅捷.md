# 概念：机制_迅捷

## 定义

**迅捷**是洛克王国世界中通过**主动轮换精灵入场**触发的特殊效果机制，允许以极低（甚至零）费用使用特定技能。

## 触发条件

| 条件 | 要求 |
|------|------|
| 入场方式 | 必须 **主动切换精灵**（不能是死亡后被动换人） |
| 技能要求 | 使用第一个**能量满足要求**且**带有迅捷标记**的技能 |

> [!warning] 关键限制
> **精灵死亡后换人入场不会触发迅捷！**
> 这意味着一旦该宠物被击杀，它的迅捷价值就彻底丧失。

## 迅捷水刃 — 雪影传说队核心联动

### 原理

```
圣羽翼王 携带 水刃
    +
翠顶夫人 也携带 水刃（同队伍中的另一只翼系）
    =
圣羽翼王的 水刃 获得 迅捷 标记
```

### 完整流程

1. **前置条件**：敌方使用了状态类技能（非直接攻击）
2. **水刃应对**：水刃"应对"到状态技能 → **费用变为 0**
3. **主动轮切**：玩家主动切换 → 圣羽翼王上场 → **触发迅捷**
4. **打出**：使用 0 费的迅捷水刃造成伤害
5. **切走**：保留圣羽翼王，下回合继续循环

### 为什么需要"应对"到状态？

- 水刃本身有能量费用
- 当水刃"应对"（counter）到敌方的状态技能时，费用降为 **0**
- 此时配合迅捷 = **免费白嫖一次伤害**

## 战术价值

| 维度 | 评价 |
|------|------|
| 资源效率 | ★★★★★（0费打伤害 = 无限续航） |
| 操作难度 | ★★★★☆（需要抓时机、预判对方出招） |
| 上限 | 极高（每回合都可能0费输出） |
| 下限 | 较低（没应对到状态则卡手） |
| 风险 | 死亡后完全失效 |
