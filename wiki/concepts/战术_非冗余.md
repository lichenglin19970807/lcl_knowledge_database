# 战术：非冗余进攻

## 非冗余 vs 冗余

| 对比项 | 冗余战术 | 非冗余战术 |
|--------|----------|------------|
| **核心机制** | 利用重复对位关系，创造进攻冗余突破 counter | 利用场上压力 + 先读，创造局部优势逐步扩大 |
| **前提条件** | 需要 1看2 或 2看3 局面 | 无特殊前提，适用范围更广 |
| **效果确定性** | 高（条件满足即可执行） | 中（依赖先读 vs 猜拳） |
| **水平敏感度** | 较低 | **高**——高分段先读退化为猜拳时效果下降 |

> 非冗余战术的底层逻辑依然是**主动创造先读场景**，但随分段提升，先读→猜拳是必然趋势。

## 一、炮台战术

### 定义

**炮台 = 能够对敌方产生极大场上压力的宠物。**

核心大 C 本身就是炮台（白金独角兽、黑猫巫师、火神等）。但炮台身份是**动态的**——具体对局、不同阶段、残局下很多宠物都能成为炮台。

### 核心逻辑

```
识别我方炮台 A
  → 利用其他宠物制造压力或直接上 A 制造压力
  → 敌方不敢留场猜拳 → 被迫切人（通常切到能 counter A 的宠物 C）
  → 先读敌方切 C → 直接切换炮台 B 安全上场
  → B vs C 的局面中 B 占优 → 扩大战果
```

### 多炮台变体

当有 2 个以上炮台时可形成配合：

```
A炮台（较肉，压力明确指向C）上场
  → 敌方被迫切 C 来针对 A
  → 先读敌方切 C → 切 B炮台（更脆但威胁更大）上场
  → B vs C → B 释放更强压力 → 更大战果
```

**关键**：C 很难对 B 造成威胁，所以切换 B 是安全的。

### 实战案例

| 步骤 | 操作 | 结果 |
|------|------|------|
| 1 | 白金独角兽上场 | 敌方只有雪影娃娃能 counter → 被迫切雪影 |
| 2 | **先读**敌方切雪影 → 直接换**火神**上场 | 下一轮变成火神 vs 雪影娃娃 |
| 3 | 火神安全叠 buff | 火神启动推队 |

## 二、捕获战术

### 定义

> 通过 A 的场上压力逼迫对手切换 B 上场防御 → **先读 B 上场** → 切出迅捷宠物 C 打迅捷技能直接重创 B。

### 为什么叫"捕获"

- 目标 B 往往是 A 的 **1看1 拦截位**
- 重创 B 后，后续 A 再次上场就无人可挡了
- 相当于"捕获"了对方的拦截手段

### 核心逻辑

```
A（大C/炮台）上场施压
  → 敌方只有 B 能 counter A → 被迫切 B
  → 先读 B 上场 → 切迅捷 C 打迅捷技能
  → B 被**重创/秒杀**
  → A 再次上场 → 敌方无 counter → A 推队
```

### 与炮台战术的区别

| 维度 | 炮台战术 | 捕获战术 |
|------|----------|----------|
| **目标** | 让己方炮台安全上场 | **重创/消灭敌方 counter 位** |
| **手段** | 先读对方切人后换自己人 | 先读对方切人后用**迅捷技能打切入** |
| **结果** | 建立有利对位 | **永久消除拦截位** |

### 实战案例

| 步骤 | 操作 | 结果 |
|------|------|------|
| 1 | 白金独角兽上场 | 敌方只有雪影娃娃能 counter → 被迫切雪影 |
| 2 | **先读**敌方切雪影 → 切**帕帕斯卡**打**迅捷钢铁洪流** | **秒杀雪影娃娃** |
| 3 | 白金独角兽再次上场 | 敌方已无 counter → 白金推队 |

## 三、场地战术

### 定义

> 通过印记、永久 buff 等**场地效果**改变原本的 counter / check 关系。
>
> 场地效果收益非常大——一旦存在场地战术，双方重心都会围绕它展开。

### 三角色模型

```
┌─────────────────────────────────────────┐
│  起点（Originator）                       │
│  发起场地效果（通常是状态类技能）           │
│  例：圆号鱼打湿润印记                      │
├─────────────────────────────────────────┤
│  终点（Beneficiary）                      │
│  场地效果的受益者                          │
│  例：利用湿润印记获得优势的宠物            │
├─────────────────────────────────────────┤
│  工兵（Engineer）                         │
│  敌方能清除场地效果的宠物                   │
│  例：尖嘴狐仙通过焚烧烙印清除湿润印记       │
└─────────────────────────────────────────┘
```

### 角色间的 Counter 关键

这是场地战术的核心博弈：

| 局面 | 结果 |
|------|------|
| **起点被工兵 counter** | 起点很难在不解决工兵的情况下强行发起场地 |
| **工兵被起点 counter** | 工兵也很难强行清除场地效果 |
| **双方互 counter** | 典型的猜拳/先读博弈场景 |

### 应用策略一：起点被工兵 counter 时

> **让终点成为能利用工兵入场的宠物。**

本质：先读敌方工兵上场清理回合 → 转换成终点上场机会。

#### 案例

```
起点：圆号鱼（湿润印记）
工兵：尖嘴狐仙（counter圆号鱼，焚烧烙印可清湿润印记）

方案A — 迅捷反制：
  终点选迅捷翼王（水刃），先读尖嘴狐仙上场清理印记
  → 迅捷水刃重创尖嘴狐仙

方案B — 属性反制：
  终点选火系宠物，先读尖嘴狐仙要上场清理
  → 火系入场 vs 尖嘴狐仙 → 尖嘴狐仙不敢清理印记（怕被克制）
```

### 应用策略二：终点被工兵 counter 时

> **需要复数终点来突破工兵**（类似冗余战术中的定向施压）

```
终点-A 上场 → 工兵 counter 上场清除印记
  → 终点-A 硬打工兵，消耗其血量和能量
  → 工兵大残 → 无法再 counter 终点-B
  → 终点-B 上场享受场地效果 ✓
```

## 四种战术的完整定位

结合 [[concepts/战术_联攻]]（冗余战术）和本页（非冗余战术），完整进攻体系如下：

```
进攻战术选择树：
│
├─ 存在 1看2 / 2看3？
│   ├─ 有复数大C？ → 定向施压（起点终点可互换）
│   ├─ 有控制手？ → 控制（固定起点终点）
│   └─ 其他？      → 地雷（待研究）
│
└─ 不存在（1看1 / 2看2）？
    ├─ 有炮台？     → 炮台战术（压出切入位→安全换炮台）
    ├─ 有迅捷+拦截位明确？ → 捕获（迅捷打切入→消除拦截位）
    ├─ 有场地能力？ → 场地（印记改变counter关系）
    └─ 都没有？     → 纯猜拳（胜率趋近50%）
```

## 阵容构建启示

1. **炮台识别**：每个 pet 在什么对局/阶段能成为炮台？提前想清楚
2. **迅捷价值**：不仅用于轮切消耗，更是捕获战术的核心工具
3. **场地意识**：如果阵容有印记能力，明确起点/终点/谁是对面的工兵
4. **非冗余的局限**：越高分段越接近猜拳，非冗余战术的效果会递减
