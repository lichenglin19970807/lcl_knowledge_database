# 实体：宠物_黑猫巫师

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 黑猫巫师 |
| 属性 | **普通**（单属性） |
| 定位 | **核爆型魔法输出 / 高威胁目标** |
| 威胁等级 | ☢️ **核弹级** — 0费技基本一击必杀 |

## 属性相性

| 关系 | 属性 |
|------|------|
| **被克制** | **武**（唯一弱点！） |
| **抵抗** | 幽 |

> [!tip] 普通系的防御优势
> **仅被武系一个属性克制！** 这是全游戏最稳定的防御属性之一。
> 抵抗幽系，且普通系攻击无克制目标 → 对手很难用属性克制打它。

## 种族值

| 生命 | 物攻 | 魔攻 | 物防 | 魔防 | 速度 |
|------|------|------|------|------|------|
| **152** | 91 | **163** | 67 | 119 | 70 |

> [!analysis] 数值画像 — 偏科型核弹
> - **魔攻163**：极高的法术输出，配合0费核弹技 = 灭团级伤害
> - **生命152 + 魔防119**：血量和魔防都不低，不是纯粹的玻璃炮
> - **物防67 + 速度70**：明显短板。物防极脆，速度慢
> - 综合定位：**低速高血量法术核弹，怕物理先手**

## 核心机制 — 特性

### 速度救生

> 若敌方技能**足够击败自己** → 回合开始时获得 **速度 +50**

这意味着：
- 当黑猫巫师处于"会被秒杀"的状态时，反而会获得速度加成
- 可能抢到先手放出核弹
- 但如果对方没有能秒杀它的技能 → 不触发

## 技能配置 — 核弹武器库

### 🔥 彗星 — 自杀式核弹

| 项目 | 内容 |
|------|------|
| 类型 | 魔法攻击 / 普通 |
| 能耗 | **0**（免费） |
| 威力 | **240**（基础极高） |
| **副作用1** | 每失去 5% 生命 → 本次威力 -10 |
| **副作用2** | 使用后**消耗全部生命**（自残） |

> [!danger] 彗星的伤害模型
> 满血时：240 威力 × 魔攻163 = **毁灭级**
> 即使扣血后衰减，基础240仍然恐怖
> 用完后自残 = 一换一的决心
>
> **对雪影传说队的影响**：除圣剑-X外几乎无人能扛

### ☢️ 魔能爆 — 资源转换核弹

| 项目 | 内容 |
|------|------|
| 类型 | 魔法攻击 / 普通 |
| 能耗 | **0**（免费） |
| 效果 | **消耗所有能力值**（buff/debuff等），消耗越高伤害越高 |

> [!danger] 为什么魔能爆不可防御？
> 它的伤害与自身消耗的能力值挂钩。
> 如果前面叠了嗜痛的双攻buff → 魔能爆伤害爆炸式增长
> **不防御基本会被秒，除非是圣剑-X这种三围怪**

### 🛡️ 嗜痛 — 减伤叠buff

| 项目 | 内容 |
|------|------|
| 类型 | 防御 / 普通 |
| 能耗 | 2 |
| 效果 | **减伤 80%**；若应对攻击 → 受到伤害时获得**双攻 +40%** |

### 🚀 羽化加速 — 迅捷启动器

| 项目 | 内容 |
|------|------|
| 类型 | 状态 / **翼** |
| 能耗 | 2 |
| 效果 | **迅捷标记**；全技能威力 **+20** |

## 两套套路详解

### 流派 A：魔能爆彗星流（阵地一波流）

```
技能组：魔能博、彗星、魔法增效、嗜痛

T1: 博弈阶段
   ├── 出 魔法增效（0费）→ 叠魔攻buff
   └── 或出 嗜痛（2费）→ 减伤80% + 等待受击叠双攻
       ↓
T2-T3: 核弹发射
   ├── 魔能爆（0费）→ 消耗所有能力 → 秒杀一只
   └── 彗星（0费）→ 240威力 → 再带走一只（自残换掉）
       ↓
结果: 稳定换掉敌方 1~2 只精灵
```

> [!tip] 这套的博弈点
> T1 是关键博弈回合：
> - 如果对方攻击 → 触发嗜痛减伤+叠双攻 → 后续魔能爆更疼
> - 如果对方也叠buff/防守 → 你也叠魔法增效 → 互飙数值
> - 无论哪种，T2-T3 的核弹都极其致命

### 流派 B：羽化连击流（游击轮切流）

```
技能组：羽化加速、嗜痛、午夜噪音/乱打、彗星（备选）

T1: 主动切换入场 → 触发羽化的迅捷
   → 放出 迅捷羽化加速（或迅捷连击技能）
   → 全技能威力+20
       ↓
T2: 嗜痛（减伤80% + 叠双攻）
   或 直接 连击技能（5段伤害）
       ↓
T3+: 午夜噪音(5连击×20) / 乱打(5连击×25)
   配合威力和双攻buff → 总伤害爆炸
       ↓
结果: 通过轮切稳定收割 1~2 只精灵
```

> [!note] 与音速犬的对比
> 音速犬靠热身4倍打单体爆发
> 黑猫巫师靠连击（5段）+ 威力buff打多段爆发
> 连击的好处是不怕闪避/护盾类效果（多段判定）

## 与雪影传说队的相性分析

### 对我方有利

| 维度 | 分析 |
|------|------|
| 仅被武克制 | 龙息帕尔（恶系）不克制普通，但武系技能可以针对 |
| 速度70极慢 | 我方大部分宠物都比它快（除了圣剑-X 65） |
| 物防67极脆 | 物理核心龙息帕尔有机会先手秒它 |
| 两套套路都需要铺垫 | 不是上来就扔核弹，有反应窗口期 |

### 对我方不利

| 维度 | 分析 |
|------|------|
| 0费核弹无法通过能耗限制 | 冻结/能耗debuff对其无效 |
| 彗星/魔能爆伤害溢出 | 除圣剑-X外全员一击即死 |
| 嗜痛减伤80% | 想快速击杀它也很难 |
| 普通系无克制弱点 | 属性层面无法利用克制关系打击 |
| 特性的速度+50 | 快被秒时反抢先手放核弹 |
