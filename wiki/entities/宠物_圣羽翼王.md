## 特性与核心机制

### 普通特性

对本精灵的技能，若队伍中其他翼系精灵携带相同技能 → 被击败时额外损失 1 点魔力

### 🌟 迅捷（核心机制）

详见 [[concepts/概念_机制_迅捷]]

- **触发条件**：通过**主动切换精灵**的方式入场时触发
- **效果**：使用第一个能量满足要求并带有"迅捷"标记的技能
- **限制**：精灵死亡后换人入场**不会**触发迅捷（必须是主动轮切）

### 迅捷水刃联动

> [!important] 关键配合
> 队伍中另有另一只翼系精灵（[[entities/宠物_翠顶夫人]]）也携带了**水刃**
> → 因此圣羽翼王的**水刃带有迅捷效果**

## 战术角色 — 核心打法

### 理想循环

```
1. 敌方使用状态技能（非攻击技能）
   ↓
2. 水刃"应对"到状态技能 → 费用变为 0
   ↓
3. 主动轮切圣羽翼王上场 → 触发迅捷
   ↓
4. 打出 0 费迅捷水刃 → 几乎无消耗造成伤害
   ↓
5. 切走 → 下次轮转再来
```

### 强力与弱点

| 优势 | 风险 |
|------|------|
| 0费水刃 = 白嫖伤害 | 如果没应对到状态技 → 水刃不免费 → 卡手 |
| 高速度保证轮切效率 | 魔防83偏低，被特殊攻击容易暴毙 |
| 持续消耗能力强 | 死亡后换人不触发迅捷，价值大减 |
